jQuery File Style

По умолчанию


// javascript
$('input[type=file]').filestyle();

Множественные файлы

  • Параметр multiple: true / false

// javascript
$('input[type=file]').filestyle({
    multiple: true
});


<!-- или data-атрибут -->
<input type="file" name="ATTACHMENT[]" data-multiple="true">

Собственные надписи

  • Параметр browseText: 'текст'
  • Параметр placeholderText: 'текст'
  • Параметр removeText: 'текст'
  •  
  • Атрибут data-browse
  • Атрибут data-placeholder
  • Атрибут data-remove

// javascript
$('input[type=file]').filestyle({
    browseText: 'Обзор',
    placeholderText: 'Выберите файл'
});


<!-- или data-атрибуты -->
<input type="file" name="ATTACHMENT" data-browse="Обзор" data-placeholder="Выберите файл">

Собственные надписи с HTML

  • Параметр browseText: 'текст / html'
  • Параметр placeholderText: 'текст / html'
  • Параметр removeText: 'текст / html'
  •  
  • Атрибут data-browse
  • Атрибут data-placeholder
  • Атрибут data-remove

// javascript
$('input[type=file]').filestyle({
    browseText: '<i class="material-icons">search</i> Обзор',
    removeText: '<i class="material-icons">delete</i>'
});


<!-- или data-атрибуты -->
<input type="file" name="ATTACHMENT"
    data-browse="<i class='material-icons'>search</i> Обзор"
    data-remove="<i class='material-icons'>delete</i>">

Локализация

  • Параметр lang: 'id языка из объекта languages'
  • Параметр languages: { 'id языка': { ... } }

Значение по умолчанию: 'ru'

Английская локализация входит в состав плагина (lang: 'en')



// javascript
$('input[type=file]').filestyle({
    lang: 'sweden'
    languages: {
        'sweden': {
            browseText: 'Välj fil',
            placeholderText: 'Ingen fil vald',
            removeText: 'Avlägsna'
        }
    }
});

Дополнительные классы

  • Параметр wrapClass: 'имя класса'
  •  
  • Атрибут class

// javascript
$('input[type=file]').filestyle({
    wrapClass: 'accent'
});


<!-- или атрибут class -->
<input type="file" name="ATTACHMENT" class="accent">

Уничтожение плагина

  • Параметр destroy

// javascript
$('input[type=file]').filestyle('destroy');
