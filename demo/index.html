jQuery-плагин File Style

По умолчанию


// javascript
$('input[type=file]').filestyle();

Множественные файлы (по одному)

  • Параметр multiple: true / false

В данном режиме после выбора одного файла создаётся дополнительное поле, позволяющее выбрать следующий файл

Каждый файл можно удалить или заменить по отдельности


// javascript
$('input[type=file]').filestyle({
    multiple: true
});


<!-- или data-атрибут -->
<input type="file" name="ATTACHMENT[]" data-multiple="true">

Множественные файлы (сразу несколько)

  • Атрибут multiple

Стандартное поведение поля для ввода с множественной загрузкой, при выборе нескольких файлов выводится их количество

Возможно сочетание с параметром multiple: true


<input type="file" name="ATTACHMENT[]" multiple>

Собственные надписи

  • Параметр browseText: 'текст / html'
  • Параметр placeholderText: 'текст / html'
  • Параметр removeText: 'текст / html'
  • Параметр multipleText: 'текст / html'
  •  
  • Атрибут data-browse
  • Атрибут data-placeholder
  • Атрибут data-remove
  • Атрибут data-multiple-text

// javascript
$('input[type=file]').filestyle({
    browseText: 'Обзор',
    placeholderText: 'Выберите файл',
    multipleText: 'Выбрано файлов: %s' // символ '%s' заменяется на количество выбранных файлов
});


<!-- или data-атрибуты -->
<input type="file" name="ATTACHMENT[]"
    data-browse="Обзор"
    data-placeholder="Выберите файл"
    data-multiple-text="Выбрано файлов: %s">

Собственные надписи с HTML

  • Параметр browseText: 'текст / html'
  • Параметр placeholderText: 'текст / html'
  • Параметр removeText: 'текст / html'
  • Параметр multipleText: 'текст / html'
  •  
  • Атрибут data-browse
  • Атрибут data-placeholder
  • Атрибут data-remove
  • Атрибут data-multiple-text

// javascript
$('input[type=file]').filestyle({
    browseText: '<i class="material-icons">search</i> Обзор',
    removeText: '<i class="material-icons">delete</i>'
});


<!-- или data-атрибуты -->
<input type="file" name="ATTACHMENT"
    data-browse="<i class='material-icons'>search</i> Обзор"
    data-remove="<i class='material-icons'>delete</i>">

Локализация

  • Параметр lang: 'id языка из объекта languages'
  • Параметр languages: { 'id языка': { ... } }

Значение по умолчанию: 'ru'

Английская локализация входит в состав плагина (lang: 'en')


// javascript
$('input[type=file]').filestyle({
    lang: 'sweden'
    languages: {
        'sweden': {
            browseText: 'Välj fil',
            placeholderText: 'Ingen fil vald',
            removeText: 'Avlägsna'
        }
    }
});

Дополнительные классы

  • Параметр wrapClass: 'имя класса'
  •  
  • Атрибут class

// javascript
$('input[type=file]').filestyle({
    wrapClass: 'accent'
});


<!-- или атрибут class -->
<input type="file" name="ATTACHMENT" class="accent">

Уничтожение

  • Параметр destroy

// javascript
$('input[type=file]').filestyle('destroy');
